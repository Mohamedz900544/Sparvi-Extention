Sparvi Student for macOS

1. فك ضغط الملف Sparvi_Student_Mac.zip.
2. افتح Terminal داخل الفولدر الناتج.
3. شغل الأمر ده مرة واحدة:

   chmod +x Start_Sparvi_Student_Mac.command

4. بعد كده افتح الملف:

   Start_Sparvi_Student_Mac.command

ملاحظات:
- أول تشغيل ممكن يأخذ وقت لأنه يثبت ملفات Python المطلوبة.
- لو macOS طلب صلاحيات Accessibility أو Input Monitoring، وافق عليها عشان مؤشر المدرس يشتغل.
- لازم يكون Python 3 مثبت على جهاز Mac.
